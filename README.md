# vue-template

## 主要功能点

* 路由自动化
* axios 的封装
* 移动、pc端的适配
* 通用的工具函数
* WebSocket 的嵌入
* Vuex / 自定义的状态管理
* 多页面配置
* element ui(表格的封装、主题、国际化等)

[参考文档](https://class-component.vuejs.org/)
